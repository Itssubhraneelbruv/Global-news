# Clickable Globe Starter

A minimal Next.js app for Vercel with a clickable 3D globe using `react-globe.gl`.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

### Option 1: GitHub + Vercel dashboard
1. Push this folder to a GitHub repo.
2. Import the repo into Vercel.
3. Deploy.

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel --prod
```
